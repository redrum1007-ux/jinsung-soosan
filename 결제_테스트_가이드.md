# 🧪 결제 테스트 가이드

## ✅ 이제 토스페이먼츠가 연동되었습니다!

`cart.html`의 "주문하기" 버튼이 실제 결제 페이지(`checkout-with-payment.html`)로 연결되었습니다.

---

## 📝 테스트 방법

### 1단계: 브라우저에서 열기

1. `index.html` 파일을 더블클릭
2. 또는 Live Server로 실행

### 2단계: 장바구니에 상품 추가

1. 메인 페이지에서 상품 클릭
2. "장바구니에 담기" 버튼 클릭

### 3단계: 결제 페이지로 이동

1. 우측 상단 장바구니 아이콘 클릭
2. "주문하기" 버튼 클릭 → **이제 결제 페이지로 이동됩니다!**

### 4단계: 결제 정보 입력

1. 배송 정보 입력:
   - 받는 분: 홍길동
   - 연락처: 010-1234-5678
   - 이메일: test@test.com
   - 주소: 서울시 강남구

2. 결제 방법 선택:
   - ✅ **신용/체크카드 (Toss Payments)** ← 기본 선택됨
   - 🏦 계좌이체 (Toss Payments)
   - 💳 신용카드 (PortOne)
   - 💛 카카오페이
   - 💚 네이버페이

3. "결제하기" 버튼 클릭

---

## 🎯 예상 동작

### 테스트 모드 (현재)

- **결제창이 열립니다** (토스페이먼츠 팝업)
- 하지만 **테스트 키**를 사용 중이므로 실제 결제는 안 됩니다
- 테스트 카드 정보로 테스트 가능

### 실제 운영 모드

1. 토스페이먼츠 가맹점 가입
2. 실제 API 키 발급
3. `payment-config.js`에 실제 키 입력
4. 실제 결제 가능

1. **네트워크 탭**: SDK가 로드되는지 확인
2. **콘솔 탭**: 에러 메시지 확인

## 🔍 문제 해결

### "TossPayments is not defined" 에러

→ SDK 로딩 실패. 인터넷 연결 확인

### 결제창이 안 열려요

→ F12 콘솔에서 에러 메시지 확인
→ `payment-config.js` 파일이 있는지 확인

### "successUrl이 잘못되었습니다" 에러

→ 정상입니다. 테스트용 URL이므로 실제로는 작동 안 함
