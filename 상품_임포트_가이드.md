# 📦 네이버 스마트스토어 상품 복사 가이드

## 🎯 목표

네이버 스마트스토어의 상품을 진성네이처푸드 쇼핑몰로 그대로 복사/등록하기

---

## ⚡ 빠른 방법: 상품 임포트 도구 사용

### 1단계: 임포트 도구 열기

`product-import.html` 파일을 브라우저에서 열기

### 2단계: 네이버 상품 페이지 열기

1. 네이버 스마트스토어 상품 페이지 접속
   예: https://smartstore.naver.com/jinsungnaturefood/products/9541611018

2. 다음 정보를 복사:
   - **상품명**: 페이지 상단의 제품명
   - **가격**: 판매가
   - **카테고리**: 상품 유형 선택
   - **상품 설명**: 상세 설명 복사

### 3단계: 이미지 복사

#### 대표 이미지

1. 대표 이미지 **우클릭**
2. "이미지 주소 복사" 클릭
3. 임포트 도구의 "대표 이미지 URL"에 붙여넣기

#### 상세 이미지 (여러 장)

1. 상세페이지 이미지들을 하나씩:
   - 우클릭 → "이미지 주소 복사"
   - 임포트 도구의 "상세 이미지 URL"에 붙여넣기
   - **줄바꿈**으로 여러 URL 구분

### 4단계: 상품 등록

"상품 등록하기" 버튼 클릭!

---

## 📝 수동 등록 (직접 복사)

### 방법 1: 관리자 페이지 사용

1. `admin.html` 열기
2. 관리자 로그인 (ID: admin, PW: admin1234)
3. "상품 등록" 클릭
4. 정보 입력 후 등록

### 방법 2: 코드에 직접 추가

`script.js` 파일에서 `initialProducts` 배열에 추가:

```javascript
const initialProducts = [
  // 기존 상품들...
  {
    id: 10, // 새로운 ID
    name: "프리미엄 완도 돌김 선물세트",
    price: 35000,
    category: "김/미역/다시마",
    image: "네이버에서 복사한 이미지 URL",
    description: "청정 완도 바다에서 자란 프리미엄 돌김",
    badge: "베스트",
    isNew: false,
    isBest: true,
  },
];
```

## 🖼️ 이미지 다운로드 방법 (선택사항)

### 네이버 이미지를 로컬에 저장하려면:

1. **이미지 다운로드**
   - 네이버 상품 이미지 우클릭
   - "다른 이름으로 이미지 저장"
   - `images/` 폴더에 저장

2. **이미지 경로 수정**

   ```javascript
   image: "images/product-10.jpg";
   ```

3. **장점**: 네이버 링크가 깨져도 안전
4. **단점**: 수동 작업 필요

## 🤖 자동화 방법 (고급)

네이버 스마트스토어는 크롤링 방지 정책이 있지만, 다음 방법으로 자동화 가능:

### 방법 1: 브라우저 확장 프로그램

1. "Web Scraper" Chrome 확장 프로그램 설치
2. 네이버 상품 페이지에서 스크래핑 설정
3. JSON으로 내보내기
4. `product-import.html`에서 일괄 등록

### 방법 2: Python 스크립트

```python
# Selenium 사용
from selenium import webdriver
import json

# 네이버 상품 페이지 접속
driver = webdriver.Chrome()
driver.get("네이버 URL")

# 정보 추출
product = {
    "name": driver.find_element("css selector", ".title").text,
    "price": driver.find_element("css selector", ".price").text,
    # ...
}

# JSON 저장
with open("product.json", "w") as f:
    json.dump(product, f)
```

## 🔧 문제 해결

### 이미지가 안 보여요

- 네이버 이미지 URL은 시간이 지나면 만료될 수 있습니다
- **해결**: 이미지를 다운로드해서 `images/` 폴더에 저장

### 상품이 메인에 안 나타나요

- 브라우저 새로고침 (Ctrl+F5)
- localStorage 확인: F12 → Application → Local Storage

### 네이버 이미지 주소 복사가 안 돼요

- 이미지를 먼저 **새 탭에서 열기**
- 주소창의 URL 복사
