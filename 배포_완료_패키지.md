## 🔧 로컬 실행 방법

### 프론트엔드만 (HTML 파일 열기)

1. `index.html` 더블클릭
2. 브라우저에서 실행

### 백엔드 포함 (풀스택)

```bash
# 1. 의존성 설치
npm install

# 2. 환경변수 설정
copy .env.example .env
# .env 파일 수정

# 3. 서버 실행
npm start

# 4. 브라우저 열기
http://localhost:5000
```

## 💳 결제 연동하기

### 단계:

1. **가맹점 가입**
   - Toss Payments: https://www.tosspayments.com
   - PortOne: https://portone.io

2. **API 키 발급받기**

3. **설정 파일 수정**
   - `payment-config.js` → 실제 키 입력

4. **결제 페이지 활성화**
   - `checkout-with-payment.html` 사용

자세한 내용: `PAYMENT_GUIDE.md` 참조
