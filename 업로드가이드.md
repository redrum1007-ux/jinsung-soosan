## 🚀 업로드 방법 (클릭 5번이면 완료!)

### 1단계: GitHub 웹사이트 열기
브라우저 주소창에 입력:
```
https://github.com/redrum1007-ux/jinsung-naturefood
```

### 2단계: 업로드 시작
- **"Add file"** 버튼 클릭
- **"Upload files"** 선택

### 3단계: 파일 선택
Windows 탐색기에서:
```
g:\내 드라이브\AI자료\안티그래비티\진성네이처푸드 쇼핑몰
```
폴더를 열고:
- 위 체크리스트의 ✅ 파일들만 선택 (Ctrl 누른 채로 클릭)
- images 폴더도 선택

### 4단계: 드래그 앤 드롭
- 선택한 파일들을 GitHub 페이지로 끌어다 놓기

### 5단계: 완료
- **"Commit changes"** 클릭
- 끝! ✨
