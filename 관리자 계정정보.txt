관리자 시스템 구현 완료:

🔑 관리자 로그인:

페이지: admin-login.html
기본 계정: ID admin / 비밀번호 admin1234
인증 후 관리자 페이지로 이동
🛠️ 관리자 기능 (admin.html):

상품 목록 탭: 전체 상품 조회 (이미지, 이름, 카테고리, 가격)
상품 등록 탭: 새 상품 추가 (카테고리 선택 포함)
수정: 각 상품의 "수정" 버튼으로 정보 변경
삭제: 각 상품의 "삭제" 버튼으로 즉시 삭제
로그아웃: 관리자 세션 종료
✨ 사용 방법:

admin-login.html 접속
admin/admin1234로 로그인
상품 관리 페이지에서 등록/수정/삭제
